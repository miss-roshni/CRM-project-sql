# CRM-project-sql
Customer Relationship Management (CRM) SQL Project
This SQL-based **CRM Database Management System** is designed to manage customer, sales, and product data efficiently.  

Key Tables
crm.customer – Stores customer information (name, email, phone, join date, sales rep).
crm.salesrepresentatives – Holds sales representative details and departments.
crm.orders – Records customer orders and total amounts.
crm.products – Contains product catalog with price and stock info.
crm.order_items – Links products to specific orders.
crm.leads – Manages lead data for potential customers.
crm.interactions – Tracks sales reps’ communication with leads.
crm.tasks – Lists assigned tasks to sales reps.

Run Scripts Sequentially
Execute CRM Project 1.0.sql to create schema and tables.
Execute CRM Project 1.1.sql to create master view.
Execute CRM Project 1.2.sql to create procedures.
Execute CRM Project 1.3.sql to insert sample data and test functionality
